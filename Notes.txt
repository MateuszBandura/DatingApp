entity framework is an object relational mapper. It basically translates our code into SQL commmands that update our tables in the database
when we add entity framework we need to make Dbcontext. Dbcontext basically acts as bridge between the domain/entity classes and database
It represents a session with the database
entity framework works with sql provider
it provides querying, change tracking, saving, concurrency, transactions, caching, built-in conventions, configurations and migrations
