observable is basically a tool to manage async data streams. It represents a stream of data that can emmit multiple values overtime
if we want to observe the observable we need to subscribe to it
due to the fact that modern angular is "zoneless", we use signal() instead of normal typescript variables incases where we want to display or use that value u somewhere
whenever signals value changes it notifies everything that uses that signal that it changes allowing the site to update without reloading for example
